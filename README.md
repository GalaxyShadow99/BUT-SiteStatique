Nous nous sommes éloignés de notre maquette initiale pour des soucis d'ergonomie et de facilité de navigation.

Le site marche bien avec une fonctionnalité de login via JavaScript, si le login ne change rien alors veuillez passer sur chrome il est possible de rencontrer des erreurs.

Préférable en thème clair, certaines couleurs ne sont pas visibles en thème sombre.

La page de connexion ayant été réalisée entièrement en JS, dû à l'absence de base de données, merci de ne pas cliquer sur une lien juste après vous être connecté (cela videra le session storage et aura pour effet de vous supprimer le rôle donné lors de votre connexion).
Pour accéder aux liens 'exclusifs' aux différents rôles, vous serez donc redirigé vers index.html et d'ici vous pourrez accéder aux pages réservées à un certain rôle.

Les horaires mentionnés dans infrastructure.html sont pûrement fictifs et ont été générés par une IA.
